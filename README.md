# Asociația TRE® România — website

Site oficial reconstruit al Asociației TRE® România (Tension & Trauma Releasing Exercises).
Static HTML/CSS/JS, găzduit pe GitHub Pages.

- 17 pagini (Acasă, Despre TRE®, Asociația, Echipă, Educație, Formări, Cercetare, Facilitatori, Evenimente, Blog, Contact)
- 68 de imagini originale importate de pe treromania.org
- Design responsive, bilingv-ready (conținut în limba română)
